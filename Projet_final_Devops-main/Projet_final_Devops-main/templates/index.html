📊 Scraping & Analyse des Annonces Immobilières

Bienvenue dans votre application de scraping et d'analyse des annonces immobilières 📌

🚀 Fonctionnalités :

  • 📥 Scraping : Récupérez les annonces en quelques clics.
  • 📊 Vue des données : Explorez les annonces brutes collectées.
  • 📈 Dashboard : Analysez les tendances du marché.

📌 Comment l'utiliser ?

  1. Allez dans la section "Scraping" et choisissez une catégorie.
  2. Lancez le scraping pour récupérer les données.
  3. Visualisez les données brutes.
  4. Analysez les tendances dans le Dashboard.
⚡ Conseil : Pour une meilleure expérience, utilisez un écran large et testez différentes catégories !
